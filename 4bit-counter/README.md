# Arduino-4bit-Counter-54161

## Requirements
1. Arduino (Leonardo was used but any model will do)
2. 54161 4bit counter chip: https://www.questcomp.com/part/4/snc54161j/382938944
3. Wires

## Code
* Code is contained within 4bit-counter.ino file.
